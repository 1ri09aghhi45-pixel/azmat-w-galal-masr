# 🇪🇬 عظمة وجلال مصر

موقع تعليمي يهدف إلى عرض عظمة مصر من خلال استكشاف أبرز المعالم السياحية والمشروعات القومية والتنمية الحديثة بأسلوب بسيط وجذاب.

---

## ✨ محتوى الموقع

يضم الموقع عدة أقسام رئيسية:

* 🏛️ **المعالم الأثرية**
  عرض لأهم المعالم السياحية مثل الأهرامات وأبو الهول وغيرها.

* 🌾 **المشروعات الزراعية**
  استعراض أبرز المشروعات الزراعية مثل الدلتا الجديدة وتوشكى.

* 🏙️ **النهضة العمرانية**
  عرض المدن الجديدة والتطور العمراني في مصر.

* ⚡ **المدن الصناعية ومصادر الطاقة**
  التعرف على التطور الصناعي ومصادر الطاقة الحديثة.

* 💻 **النهضة الرقمية**
  إبراز التحول الرقمي والتقدم التكنولوجي في مصر.

---

## 🎯 هدف المشروع

* نشر الوعي بإنجازات الدولة المصرية
* تقديم المعلومات بشكل مبسط للطلاب
* تنمية مهارات تصميم المواقع باستخدام HTML و CSS
